rest-amqp

Rest API using amqp protocol for NodeJS. Tested on RabbitMQ

View on GitHub Download .zip Download .tar.gz

"rest-amqp is Rest API using amqp protocol for NodeJS. Build micro-service over it"

npm install --save rest-amqp

Example: var restAMQP = require('../index').init(); var debug = require('debug')('rest-amqp-sample');

restAMQP.get('/hi/:id', function (req, res) {
   res.send("Hello rest-amqp");
});

restAMQP.listen();
